## Common Commands

| Command   | Description                            |
|----------|----------------------------------------|
| SET      | Set key-value pair                     |
| GET      | Get value of a key                     |
| DEL      | Delete key                             |
| EXPIRE   | Set TTL in seconds for a key           |
| TTL      | Check remaining TTL of a key           |
| PING     | Health check                           |
| PUBLISH  | Publish message to subscribers         |

Refer to Python and Node.js sections for usage examples.
