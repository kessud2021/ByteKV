# Welcome to ByteKV Documentation

ByteKV is a fast, TCP-based in-memory key-value store with a Redis-like RESP protocol.  
It provides official **Python** and **Node.js** clients.

- Supports `SET`, `GET`, `DEL`, `EXPIRE`, `TTL`, `PING`, `PUBLISH`
- Works with Python 3.7+ and Node.js 14+
- Easy to integrate into Python or Node.js applications

Explore the documentation to get started!
